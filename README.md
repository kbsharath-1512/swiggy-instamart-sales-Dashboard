# 🛒 Swiggy Instamart Sales Analytics Dashboard

## 🚀 Project Overview

An interactive Power BI dashboard developed to analyze Swiggy Instamart sales performance, customer ratings, order trends, and shop-level insights.

The dashboard provides a comprehensive view of business performance through dynamic filters, KPIs, and visual analytics, helping stakeholders make data-driven decisions.

---

## 📊 Dashboard Highlights

### 📌 Interactive Filter Panel
- Year-wise filtering (2021–2024)
- Shop Size filtering (High, Medium, Small)
- City Tier filtering (Tier 1, Tier 2, Tier 3)
- Clear All Filters functionality

### 📈 Key Performance Indicators (KPIs)
- Total Sales: **1.20M**
- Total Orders: **8.52K**
- Average Rating: **3.92**
- Average Item Weight: **12.86**

### 🏪 Shop Analysis
- Sales by Shop Type
- Sales by Shop Size
- Orders by Shop Type
- Shop Performance Comparison

### 🌆 City Analysis
- Total Sales by City Tier
- Total Orders by City Tier
- Tier-wise Performance Analysis

### 📅 Trend Analysis
- Shops Opening Year-wise
- Historical Sales Performance
- Order Growth Analysis

### 🥫 Product Category Analysis
- Total Sales by Category
- Top Performing Categories
- Category-wise Revenue Contribution

### ⭐ Customer Insights
- Average Customer Ratings
- Rating Performance Analysis
